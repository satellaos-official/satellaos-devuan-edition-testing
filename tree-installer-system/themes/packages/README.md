# Fluent GTK Theme (.deb Package)

> **This repository is NOT the original Fluent GTK Theme project.**

This repository provides a pre-packaged **`.deb`** version of the **Fluent GTK Theme** for Debian-based Linux distributions.

Its primary purpose is to **reduce installation time** and simplify deployment during the installation of my Linux distribution. Instead of cloning the repository and running the installation script, the theme can be installed directly as a Debian package.

## Original Project

All credits for the Fluent GTK Theme belong to **Vince Liuice**.

**Official repository:**
https://github.com/vinceliuice/Fluent-gtk-theme

If you enjoy this theme, please consider starring the original repository and supporting its author.

## Why does this repository exist?

This project exists solely for convenience and faster installation.

The goals are:

* Faster installation during OS setup
* Easier integration into Debian-based distributions
* Offline installation support
* Simplified package management using `apt`/`dpkg`

This repository **does not attempt to replace or fork the original project**.
